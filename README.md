# phpftp

Web based FTP client.

## Testing

I use [DLP Test](https://dlptest.com/ftp-test/) to test the program. 

## Features 

**FTP**

- [x] login
- [x] logout
- [x] list files in current directory
- [ ] change directory
- [ ] create directories
- [ ] upload files
- [ ] delete files
- [ ] edit text files
- [ ] download files
- [ ] preview files
- [ ] change permissions
- [ ] view permissions

**Other**

- [ ] automatically check if API host runs PHP (the JS only needs to be created)
